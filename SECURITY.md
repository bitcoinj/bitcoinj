# Security Policy

## Supported Versions

We will accept and respond to vulnerability reports for the supported releases below as well as for any issues discovered on the `master` branch.

| Version  | Supported          |
| -------  | ------------------ |
| 0.17.x   | :white_check_mark: |
| < 0.17.x | :x:                |

## Reporting a Vulnerability

To report a vulnerability please use "Report a vulnerability" on the bitcoinj [Security](https://github.com/bitcoinj/bitcoinj/security) page.
